# evaltmpl

## Desc

eval any go template

## Installation

    $ go get github.com/sheercat/evaltmpl

## Synopsis

    $ ./evaltmpl

or

    $ GOOS=linu GOARCH=amd64 go build evaltmpl.go
    $ docker build -f Dockerfile.local -t evaltmpl .
    $ docker run -it --rm -p 3000:3000 evaltmpl
